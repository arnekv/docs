## Create a canvas[​](#create-canvas "Direct link to heading")

To create a canvas:

1. Navigate to ***Industrial tools*** > ***Canvas***.

2. Select ***+ Create canvas*** to start a new canvas.

   ![Canvas](\images\cdf\explore\\-cognite\docs-portal-images\1.0.0\images\cdf\industrial_tools\canvas.svg)

➀ **Toolbar** — hover over each tool to see a description.

➁ **Settings** — select canvas **settings**, add or remove **labels**, **save as PDF**, **lock**, or **duplicate** a canvas.

➂ **Menubar** — select an item on the canvas to show **relevant options** for the item, for example, to open it in search.

➃ **Agents** — open an **AI agent** to work with the canvas.

⑤ ***Add data*** — find and **select data** to add to the canvas. You can also **drag and drop** local files and images directly to the canvas.

## Engineering diagrams[​](#diagram-options "Direct link to heading")

Add engineering diagrams and related data to a canvas.

![Canvas engineer diagram](\images\cdf\explore\\-cognite\docs-portal-images\1.0.0\images\cdf\industrial_tools\canvas_engineering_diagram.svg)

➀ **Add related data** — click a purple or orange link and the plus sign (+) to add related data, such as engineering diagrams, files, and time series.

➁ **View live time series data** — pin a time series to the canvas, set threshold values, and use colors to visualize any breaches.

## Set threshold rules *[(beta)](/cdf/product_feature_status)*[​](#add-rules "Direct link to heading")

Use rules to monitor items on a canvas. For instance, to keep track of pump values, corrosion rates, or operating temperatures.

1. Add or select an item on a canvas, for example, a sticky note, or a connection.

2. Click ![ ](\images\cdf\explore\\-cognite\docs-portal-images\1.0.0\images\cdf\explore\lightning.png) > ***Create rule***.

3. Give the rule a name and optionally, create a rule group.

4. Select the ***Trigger***, ***Criteria***, and ***Formatting*** for the rule.

## Navigate[​](#navigate "Direct link to heading")

To navigate on the canvas:

* To **pan**, use the `Spacebar` and the mouse, or use two or more fingers on a touchpad.

* To **move** items, select and drag.

* To **zoom**, use the mouse scroll wheel, or pinch and stretch on a touchpad.
